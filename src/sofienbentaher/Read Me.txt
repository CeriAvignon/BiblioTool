
************************
dblp.search.engine
************************

Cette classe permet la recherche sur la base de donnée en ligne DBLP avec son propres API.

*****Les entrées :

     -Le nom d'auteur: La méthode public InputStream searchAuthor(String mot) permet
de faire la recherche avec cette entrer.

     -Un mot clé: La méthode public InputStream searchMotCle(String mot) permet
de faire la recherche avec cette entrée

*****Les sorties:

 Ce classe résulte 4 variables :

  - Le nom d'auteur avec la méthode public String[] getAuthor(JSONObject obj)
  - L'URL d'un article avec la Méthode public String getUrl(JSONObject obj)
  - La date ou bien L'année du publication avec la méthode public String getYear(JSONObject obj)
  - Le Titre de l'article avec la méthode public String getTitle(JSONObject obj)
